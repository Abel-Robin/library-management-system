# 📚 Library Management System

A simple command-line **Library Management System** built with **Python**. This project demonstrates CRUD (Create, Read, Update, Delete) operations using CSV file storage.

## 🚀 Features

* ➕ Add new books
* 📖 View all books
* 🔍 Search books by Book ID, Title, or Author
* ✏️ Update book information
* 🗑️ Delete books
* 💾 Data stored in a CSV file

## 🛠️ Technologies Used

* Python 3
* CSV Module
* File Handling

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/Abel-Robin/Library-Management-System.git
```

2. Navigate to the project folder:

```bash
cd Library-Management-System
```

3. Run the program:

```bash
python main.py
```
